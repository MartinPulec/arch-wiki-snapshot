Partclone, как и известный Partimage, может использоваться для резервного копирования и восстановления раздела с учётом только используемых блоков.

Установка

Установите пакет partclone.

Использование Partclone с разделом ext4

Без сжатия

Создание резервной копии без сжатия: 

# partclone.ext4 -c -s /dev/sda1 -o ~/image_sda1.pcl

Восстановление: 

# partclone.ext4 -r -s ~/image_sda1.pcl -o /dev/sda1

Со сжатием

Создание резервной копии со сжатием: 

# partclone.ext4 -c -s /dev/sda1 | gzip -c > ~/image_sda1.pcl.gz
Примечание: Для наилучшего сжатия можно использовать gzip -c9 или поэкспериментировать с другими инструментами сжатия.

Восстановление: 

# zcat ~/image_sda1.pcl.gz | partclone.ext4 -r -o /dev/sda1
