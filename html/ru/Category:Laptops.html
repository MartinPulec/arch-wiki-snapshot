Help
From ArchWiki

Эта категория содержит статьи про установку и настройку Arch Linux на ноутбуках. Специфичные для определённых моделей статьи должны быть категоризированы в соответствующую подкатегорию производителя для избежания помойки.

Примечание:
  • Для общих инструкций по установке Arch Linux вы всегда должны следовать Installation guide и использовать соответствующие статьи по ноутбукам для того, чтобы быть в курсе специфики оборудования. Для послеустановочных инструкций следуйте General recommendations и Laptop мета страницам с соответствующим контентом, который сопровождайте специфичными для оборудования инструкциями, которые могут содержать статьи в этой категории.
  • Если вы хотите добавить информацию, сначала подумайте куда её лучше поместить. В частности, не дублируйте существующие инструкции из других статей на ArchWiki про ноутбуки. Вместо этого сделайте упор на специфичные для оборудования настройки и приводите ссылки на существующие инструкции, либо улучшите их, если ваш вклад может быть применим к другому оборудованию.
  • Если у вас малая заметка, вы можете добавить запись в соответствующую статью в HCL/Laptops.
  • Если вы хотите создать новую статью, есть два примера:
    • Пример статьи про ноутбук с минимальным дублированием контента и акцентирующей внимание на конкретном оборудовании ASUS N550JV.
    • Другой пример - это Acer C720 Chromebook, которая описывает специфичные для оборудования инструкции и опирается на Chrome OS devices как на отдельную мета статью, потому что категория устройства сама по себе очень специфична и требует очень специфичных действий при установке.
