Django — высокоуровневый это веб-фреймворк Python, который следует архитектурному шаблону модель-представление-шаблон (MVT).

Установка

Установите пакет python-django. Документацию можно скачать с помощью пакета python-django-docsAUR.

Драйвер базы данных

Для Django доступны различные серверные базы данных:

  • Для работы с PostgreSQL установите пакет python-psycopg2.
  • Если вы хотите использовать MySQL, то установите пакет python-mysqlclient.

Применение

Для создания проекта на Django и начала работы используйте команду django-admin: 

$ django-admin startproject mysite

Команда создаст каталог mysite в текущем каталоге. Она также создаст скрипт manage.py, который позволит взаимодействовать с вашим проектом.

Больше информации можно увидеть в официальном руководстве по Django и документации по Django.
