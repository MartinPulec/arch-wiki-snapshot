Bottles - это менеджер префиксов Wine, написанный на Python с использованием фреймворка GTK. Может использоваться для создания и управления префиксами Wine, а также для автоматической установки различных оболочек Wine, зависимостей Windows и установки некоторых приложений под Windows. С его помощью можно переопределять DLL-файлы Windows внутри префикса и управлять переменными окружения для сессий Wine.

В Bottles можно запускать нативные Windows-приложения и игры с производительностью в большинстве случаев близкой к нативной. Поддерживает запуск приложений в "песочнице".

Примечание: Разработчики Bottles настоятельно рекомендуют пользователям устанавливать Bottles через Flatpak, так как он используется для "песочницы".

Установка

Установите Bottles из Flatpak с помощью следующей команды: 

$ flatpak install bottles

Использование

Bottles имеет подробное руководство по использованию.
