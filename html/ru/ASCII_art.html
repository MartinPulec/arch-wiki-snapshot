Cowsay

Примеры использования cowsay и fortune:

Стандартная корова c fortune

  • В сочетании с cowsay:
$ cowthink $(fortune)

 ________________________________________ 
( The best cure for insomnia is to get a )
( lot of sleep. -W.C. Fields             )
 ---------------------------------------- 
        o   ^__^
         o  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||

ASCII-изображения генерируются текстовыми файлами .cow, расположенными в /usr/share/cows, и все темы могут быть перечислены с помощью команды cowsay -l. Эти файлы можно редактировать по своему усмотрению; пользовательские изображения также можно создать с нуля или найти в сети. Самый простой способ создать пользовательский файл cow - использовать существующий в качестве шаблона. Чтобы проверить пользовательский файл: 

$ cowsay -f /путь/к/файлу/имя_файла $(fortune)

Случайная говорящая корова с fortune

$ fortune -c | cowthink -f $(find /usr/share/cows -type f | shuf -n 1)

Случайная кастомизируемая корова с fortune

Сложные команды можно объединить в цепочку, чтобы получить детализированный ASCII-арт, как, например, здесь.

  • Например, случайная корова со случайным выражением лица, произносящая случайное предсказание:
Note Случайно выбранная корова (на самом деле жаба), выбранная из базы данных пользовательского ASCII-арта.

Ponysay

Для получения полного 256-цветного cow-like арта используйте ponysay (версия 3.0 содержит 422 пони). Синтаксис тот же, используйте ponysay message, чтобы сказать что-то, и ponysay -l, чтобы получить полный список пони. Чтобы выбрать пони для отображения, выполните ponysay --pony x "message", где x - пони. Чтобы создать больше пони, используйте util-say-gitAUR и храните их в ~/.local/share/ponysay/ponies и ~/.local/share/ponysay/ttyponies/ для настольных компьютеров и TTY, соответственно.
