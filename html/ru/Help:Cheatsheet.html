Добавление страницы в категорию

Категории следует помещать в начале кода страниц.
 [[Category:Category name (Русский)]] Название категории будет отображено на панели внизу страницы после сохранения или при предварительном просмотре.
Работает везде в тексте (HTML)
Описание Код Результат
Перечёркнутый текст <s>перечёркнутый</s> перечёркнутый
Подчёркнутый текст <u>подчёркнутый</u> подчёркнутый
Текст в верхнем и нижнем индексе нормально <sub>внизу</sub> <sup>наверху</sup> нормально внизу наверху
Работает только в начале строк
Описание Код Результат
Блок кода
Добавьте один или несколько пробелов в начале каждой строки кода.
Смотрите также Help:Стиль#Форматирование кода.
  $ echo Hello World 
$ echo Hello World

Перенаправление на другую страницу

Директива перенаправления должна быть размещена в начале первой строки.
Смотрите также Help:Редактирование#Перенаправления.
 #REDIRECT [[Целевая страница]] ↳ Целевая страница

Заголовки разделов

Содержание генерируется автоматически для статей с четырьмя или более разделами.
Не используйте заголовки = уровня 1 =.
Смотрите также Help:Effective use of headers.
 
== Уровень 2 ==
=== Уровень 3 ===
==== Уровень 4 ====
===== Уровень 5 =====
====== Уровень 6 ======

Уровень 2

Уровень 3

Уровень 4

Уровень 5
Уровень 6
Горизонтальная разделительная линия ----
Маркированный список 
* Раз
* Два
** Два точка один
* Три
  • Раз
  • Два
    • Два точка один
  • Три
Нумерованный список 
# Раз
# Два
## Два точка один
# Три
  1. Раз
  2. Два
    1. Два точка один
  3. Три
Список определений 
; Первый термин: Определение 1
; Второй термин: Определение 2
Первый термин
Определение 1
Второй термин
Определение 2

Отступ текста

В основном это используется для отступов на страницах обсуждений, визуально отделяя ответы друг от друга.
 
без отступа
:одинарный отступ
::двойной отступ
:::тройной отступ

без отступа

одинарный отступ
двойной отступ
тройной отступ
