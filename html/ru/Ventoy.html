Ventoy — бесплатный инструмент с открытым исходным кодом для создания накопителей, позволяющих выбрать и загрузить любые EFI, IMG, ISO, VHD(x), или WIM файлы, хранящиеся на перезаписываемом разделе диска.

Поддерживается большинство операционных систем (Windows/WinPE/Linux/ChromeOS/Unix/VMware/Xen...), смотрите полный список.

Установка

Установите пакет ventoy-binAUR или пакет ventoyAUR.

Использование

Важно: используемый накопитель будет стёрт и все данные будут потеряны после установки.

Для настройки носителя существует три утилиты:

  1. /opt/ventoy/Ventoy2Disk.sh, который является является shell скриптом для запуска с командной строки.
  2. /opt/ventoy/VentoyGUI.x86_64, который является графическим приложением. xauth или похожее приложение необходимо для эскалации до root, если приложение запущено не с правами root.
  3. file:///opt/ventoy/WebUI/index.html открывается с web-браузером.

Те же утилиты могут использоваться для обновления установки Ventoy на накопитель.

Ventoy создаёт два раздела на накопителе. Их названия по умолчанию: Ventoy and VTOYEFI. Раздел Ventoy используется для хранения загрузочных образов (iso файли), и другие данные. VTOYEFI используется для бинарных файлов Ventoy.

Для добавления образа, с которого вы можете загрузиться, смонтируйте первый раздел и скопируйте образы.

Использование без установки

Если вы хотите только создать мультизагрузочный накопитель, вы можете только скачать готовый исполнительный файл на странице релизов на GitHub. Вам необходим dosfstools для установки, потому что раздел vfat создан с использованием mkfs.vfat. После верификации sha256sum и распаковки, вы можете отобразить страницу помощи включенного CLI [1], запустив shell скрипт без каких-либо аргументов: 

# ./ventoy-VERSION/Ventoy2Disk.sh

После этого можно обновить диск до будущих версий, выполнив аналогичную процедуру. Загрузка НОВОЙ ВЕРСИИ, проверка суммы sha256sum, распаковка, запуск 

# ./ventoy-NEWER-VERSION/Ventoy2Disk.sh

и следование появляющемуся на экране справочному сообщению.

В качестве альтернативы оболочке .sh скрипта описанной здесь, GTK (Русский)/qt (Русский)[2], и webUI[3], также включены.

Советы и рекомендации

Какая версия уже установлена?

Утилитой Ventoy2Disk.sh, упомянутой ранее, запустите процесс обновления (-u) и не утверждайте его. Утилита сообщит, с какой версии она собирается обновиться, а затем попросит подтверждения.

Проверка целостности файлов

Как описано в исходной документации, встроенная утилита проверки контрольных сумм позволяет проверить целостность файлов на диске.
