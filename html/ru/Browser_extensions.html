Browser extensions (Русский)

Appearance
From ArchWiki
Состояние перевода: На этой странице представлен перевод статьи Browser extensions. Дата последней синхронизации: 25 ноября 2022. Вы можете помочь синхронизировать перевод, если в английской версии произошли изменения.

В этой статье перечислены некоторые расширения для браузеров, доступные для Firefox и/или Chromium.

Установка

Расширения Firefox можно установить с сайта addons.mozilla.org; управление расширениями выполняется через страницу about:addons.

Расширения Chrome можно установить из Chrome Web Store; управление расширениями выполняется через страницу chrome://extensions/.

Некоторые расширения для Firefox также доступны в официальных репозиториях и в AUR.

Чтобы упростить обслуживание, в этой статье нет ссылок на страницы магазинов или пакеты расширений в AUR. Если пакет недоступен, читателям рекомендуется получать расширения через официальные сайты по указанным ссылкам.

Конфиденциальность

Смотрите также Firefox/Privacy и Chromium#Security.

Совет Не рекомендуется устанавливать все эти расширения вместе. Это может быть контрпродуктивно, так как они конфликтуют друг с другом, и это всё равно не повысит безопасность.

Блокировщики контента

  • uBlock Origin — Лёгкий, эффективный блокировщик, который не требует много памяти и процессора. Он поставляется с несколькими списками фильтров, готовыми к использованию из коробки (включая EasyList, Peter Lowe, несколько списков фильтров вредоносных программ). Ведущий разработчик uBlock форкнул проект и создал uBlock Origin. По состоянию на июль 2015 года бо́льшая часть разработки ведётся в uBlock Origin, и кодовые базы существенно расходятся.
https://github.com/gorhill/uBlock/ || firefox-ublock-origin, Chromium
  • Adblock Plus — Было популярным расширением для блокировки рекламы. Сейчас, когда оно специально не блокирует некоторые объявления [1], возможно, лучше использовать другой блокировщик, например uBlock Origin.
https://adblockplus.org/ || firefox-adblock-plus, Chromium

Расширенный контроль

  • uMatrix — Форк HTTP Switchboard. Позволяет выборочно блокировать Javascript, плагины или другие ресурсы и контролировать сторонние ресурсы. Он также обладает широкими возможностями обеспечения конфиденциальности, такими как маскировка user-agent, блокировка ссылок и так далее. Он фактически заменяет NoScript и RequestPolicy. О различных способах его использования читайте в old HTTP Switchboard wiki.
https://github.com/gorhill/uMatrix || firefox-umatrixAUR, Chromium
  • ScriptSafe — Предоставляет пользователям контроль и более безопасный просмотр сайтов, подчёркивая при этом простоту и интуитивность. В связи с характером этого расширения оно будет ломать большинство сайтов! Оно разработано для того, чтобы со временем обучаться на сайтах, которые вы разрешаете.
https://github.com/andryou/scriptsafe || Firefox, Chromium
  • NoScript — Отключает JavaScript и Flash на любом сайте, не внесённом пользователем в белый список. Это расширение защитит вас от эксплуатации уязвимостей, не позволяя ни одному сайту, кроме доверенных (например, вашего банка или веб-почты), выполнять свой код. После установки вы можете изменить параметры NoScript, щёлкнув его значок на панели инструментов или щёлкнув правой кнопкой мыши на странице и перейдя к NoScript. После этого у вас будет возможность включить/выключить скрипты для текущей страницы, а также для любых сторонних скриптов, на которые ссылается страница. Кроме того, вы можете временно включить скрипты только для текущего сеанса. Помните, что многие современные сайты используют скрипты для вёрстки, поэтому содержимое может выглядеть иначе. Например, проблемы с рендерингом из-за отсутствия шрифтов может произойти на сайтах, которые подгружают шрифты на лету с помощью скриптов, которые заблокировал NoScript.
    Важно По умолчанию NoScript разрешает JavaScript-код некоторых корпораций, что может вам не понравиться. Это влияет на вашу конфиденциальность. Чтобы изменить это, перейдите в Настройки > Разрешения для сайтов и установите значение НЕДОВЕРЕННЫЕ для всех сайтов, которым вы не доверяете. В Tor Browser при использовании в более безопасных режимах (где используется NoScript) это уже сделано изначально.
https://noscript.net/ || firefox-noscript
  • ScriptBlock — Аналогичен NoScript, который является расширением Firefox. Оба расширения блокируют выполнение веб-сайтом любого вида JavaScript. Однако ScriptBlock имеет более простой дизайн и его легче использовать. Он блокирует JavaScript по умолчанию. Вы можете временно или навсегда разрешить JavaScript. Как только вы разрешаете их запуск, он разрешает выполнение всех JavaScript на этой странице, поэтому, возможно, стоит использовать ScriptBlock в сочетании с Privacy Badger. Также стоит проверить его белый список по умолчанию, который вы, возможно, захотите изменить.
https://github.com/compvid30/scriptblock || Chromium
  • Cookie AutoDelete — Удаляет файлы cookie сразу после закрытия вкладки. Поддерживает автоматический и ручной режимы очистки кук. (Поддержка очистки LocalStorage была добавлена в версии 2.1, но только для Firefox версий 58+. В этом же выпуске добавлена поддержка first party isolation, но только для Firefox версий 59+).
https://github.com/Cookie-AutoDelete/Cookie-AutoDelete || Firefox, Chromium
  • Vanilla Cookie Manager — Менеджер белых списков файлов cookie, который автоматически удаляет нежелательные файлы cookie. Куки могут использоваться для аутентификации, хранения предпочтений сайта или чего-либо ещё, что можно сохранить в виде текстовых данных. К сожалению, они также могут использоваться для слежки за вами. Вы можете полностью отключить все куки или отключить только сторонние куки. Но это также приведёт к отключению полезных кук, на которые полагаются многие веб-приложения (например, Google Mail или Calendar). В Vanilla вы можете выбрать, какие файлы cookie вы хотите сохранить в белом списке. Все нежелательные файлы cookie удаляются автоматически (или вручную, по вашему желанию).
https://github.com/laktak/vanilla-chrome || Chromium

Автоматические блокировщики трекеров

  • Privacy Badger — Отслеживает сторонние трекеры. Он блокирует трекеры, как только они появляются на различных сайтах. Он не задумывался для блокировки рекламы, но, поскольку реклама полагается на отслеживание, на неё это тоже влияет. Для получения дополнительной информации о механизме смотрите FAQ.
https://www.eff.org/privacybadger || firefox-extension-privacybadgerAUR, Chromium
  • Disconnect — Стремится помешать 2000 сторонним сайтам отслеживать пользователя. Он шифрует данные, отправляемые на популярные сайты, и утверждает, что веб-страницы загружаются на 27 процентов быстрее. Disconnect показывает своим пользователям в режиме реального времени, сколько попыток слежения со стороны Google, Twitter, Facebook и других сайтов было остановлено. Он разделяет попытки отслеживания на рекламные, аналитические, социальные и контентные, что позволяет легко отслеживать, как за человеком следят. Disconnect также может остановить side-jacking, который использует украденные куки для кражи персональных данных. Он прост в использовании и хорошо поддерживается. Firefox получил функцию, основанную на списке Disconnect; смотрите Firefox/Privacy#Tracking protection.
https://disconnect.me/ || Firefox, Chromium

Генераторы шума

  • AdNauseam — Лёгкое расширение для браузера, сочетающее в себе программный инструмент и художественное вмешательство для борьбы с отслеживанием рекламными сетями. AdNauseam работает как блокировщик рекламы (построен на базе uBlock-Origin), по-тихому имитируя клики по каждой заблокированной рекламе, сбивая с толку трекеры в отношении реальных интересов пользователя.
https://adnauseam.io/ || Firefox, Chromium
  • TrackMeNot — Периодически отправляет случайные поисковые запросы популярным поисковым системам, что помогает спрятать ваши настоящие запросы в облаке «призрачных» запросов.
https://cs.nyu.edu/trackmenot/ || Firefox, Chromium

Разное

  • HTTPS Everywhere — Заставляет устанавливать соединение по HTTPS вместо HTTP везде, где это возможно. HTTPS Everywhere будет автоматически настроен и включен после перезапуска Firefox. Информацию о том, как настроить собственные правила для различных веб-сайтов, можно найти на официальном сайте. HTTPS Everywhere не добавляет поддержку HTTPS магическим образом для каждого сайта в Интернете. Сайт должен поддерживать HTTPS, а HTTPS Everywhere должен иметь набор правил, настроенный для этого сайта.
https://www.eff.org/https-everywhere || firefox-extension-https-everywhereAUR[ссылка недействительна: package not found], Chromium
  • Decentraleyes — Защищает вас от слежки через «бесплатные» CDN. Он предотвращает попадание большого количества запросов в такие сети, как Google Hosted Libraries, и вместо них отдаёт локальные файлы, чтобы сайты не ломались. Дополняет обычные блокировщики контента.
https://decentraleyes.org/ || firefox-decentraleyes, Chromium
  • LocalCDN — Форк Decentraleyes с поддержкой большего количества библиотек и функций
https://localcdn.org/ || Firefox
  • CanvasBlocker — Блокирует или подделывает JS-API для работы с <canvas>, чтобы предотвратить Canvas-Fingerprinting. Firefox имеет встроенную функцию защиты от фингерпринтинга, которую можно включить, установив privacy.resistFingerprinting в true в about:config.
https://github.com/kkapsner/CanvasBlocker/ || Firefox
  • Privacy Settings — Предоставляет панель инструментов для лёгкого изменения встроенных настроек конфиденциальности браузера.
https://add0n.com/privacy-settings.html || Firefox, Chromium

Кастомизация сайтов

Внешний вид и содержимое веб-сайтов можно модифицировать с помощью пользовательских стилей и пользовательских скриптов.

  • Stylus — Менеджер пользовательских таблиц стилей, форк Stylish.
https://add0n.com/stylus.html || firefox-stylusAUR, Chromium
  • Violentmonkey — Свободный менеджер пользовательских скриптов.
https://violentmonkey.github.io/ || Firefox, Chromium
  • Tampermonkey — Проприетарный менеджер пользовательских скриптов.
https://tampermonkey.net/ || Firefox, Chromium
  • Dark Reader — Инвертирует яркость веб-страниц для снижения усталости глаз при просмотре веб-страниц.
https://darkreader.org/ || firefox-dark-reader, Chromium
  • Toggle Website Colors — Заменяет цвета на выбранные пользователем.
https://github.com/M-Reimer/togglewebsitecolors || Firefox

Сочетания клавиш

Расширения, предоставляющие сочетания клавиш в стиле vi.

  • Vimium — Позволяет просматривать веб-страницы без использования мыши, имеет экспериментальную версию для Firefox.
https://github.com/philc/vimium || Firefox, Chromium
  • Vim-Vixen — Vim-подобная работа в Firefox.
https://github.com/ueokande/vim-vixen || Firefox
  • Saka Key — Позволяет просматривать веб-страницы без использования мыши, ориентируется на доступность.
https://key.saka.io/[устаревшая ссылка 2024-07-30 ⓘ] || Firefox, Chromium
  • Krabby — Позволяет просматривать веб-страницы без использования мыши, вдохновлён Kakoune.
https://krabby.netlify.app || Chromium, Firefox
  • Tridactyl — Заменяет механизм управления Firefox на механизм, созданный по образцу Vim.
https://github.com/cmcaine/tridactyl || firefox-tridactyl
  • wasavi — Может преобразовывать текстовые области в редакторы Vi.
https://github.com/akahuku/wasavi || Firefox, Chromium

Редактирование текста с помощью внешнего текстового редактора

Расширения для редактирования <textarea> в системных текстовых редакторах:

  • Textern — Расширение для редактирования текста в вашем любимом внешнем редакторе, требует Python скрипт, доступен как firefox-extension-textern-native-gitAUR.
https://github.com/jlebon/textern || Firefox
  • withExEditor — Просмотр исходного текста, выделение и редактирование текста с помощью внешнего редактора, требует Node.js.
https://github.com/asamuzaK/withExEditor || Firefox, Chromium
  • GhostText — Использование внешнего текстового редактора для написания текста в браузере. Всё, что вы набираете в редакторе, будет мгновенно обновляться в браузере (и наоборот). Имеет плагины для Vim, Emacs, Neovim и Visual Studio Code.
https://github.com/GhostText/GhostText || Firefox, Chromium
