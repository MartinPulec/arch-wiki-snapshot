GNU — это операционная система и обширная коллекция свободных компьютерных программ. GNU — это рекурсивный акроним, означающий «GNU's Not Unix!».

Поскольку ядро GNU, Hurd, всё ещё не готово [1], программы GNU обычно используются с ядром Linux. Arch Linux — это дистрибутив GNU/Linux, использующий программы GNU, такие как командная оболочка Bash, GNU coreutils, GNU toolchain и множество других утилит и библиотек. Эта страница не пытается перечислить все почти 400 пакетов GNU, а лишь выделяет некоторые из них.

Texinfo

Документация программ GNU пишется с использованием синтаксиса Texinfo. Вы можете просматривать документы Info с помощью программы info, доступной в пакете texinfo.

Хотя для большинства программ GNU также есть man-страницы, документация Info, как правило, более полная.

Базовая система

  • GRUB — Загрузчик.
https://www.gnu.org/software/grub/ || grub
  • Bash — Командная оболочка, совместимая с sh, которая включает в себя полезные функции из Korn shell (ksh) и C shell (csh).
https://www.gnu.org/software/bash/ || bash
  • Coreutils — Основные утилиты операционной системы GNU для работы с файлами, оболочкой и текстом.
https://www.gnu.org/software/coreutils/ || coreutils
  • gzip — Формат файла и программа для сжатия и распаковки.
https://www.gnu.org/software/gzip/ || gzip
  • tar — Программа для работы с tar-архивами.
https://www.gnu.org/software/tar/ || tar

Инструментарий

Почти все инструменты из GNU toolchain входят в состав группы base-devel, кроме glibc (требуется для base) и GDB.

  • GNU make — Утилита GNU make для управления группами программ.
https://www.gnu.org/software/make || make
  • GCC — The GNU Compiler Collection — фронтенды для C и C++.
https://gcc.gnu.org/ || gcc
  • glibc — Реализация стандартной библиотеки C от GNU.
https://www.gnu.org/software/libc/ || glibc (required by base)
  • GNU Binutils — Набор программ для сборки и манипулирования бинарными и объектными файлами. Включает в себя ld.
https://www.gnu.org/software/binutils/ || binutils
  • GNU Bison — Генератор парсеров общего назначения.
https://www.gnu.org/software/bison/bison.html || bison
  • GNU m4 — Препроцессор макросов.
https://www.gnu.org/software/m4/ || m4
  • GDB — Отладчик.
https://www.gnu.org/software/gdb/ || gdb

Система сборки

Википедия:

Autotools, или система сборки GNU, — это набор программных средств, предназначенных для поддержки переносимости исходного кода программ между UNIX-подобными системами.
  • GNU Autoconf — Инструмент для автоматической настройки исходного кода.
https://www.gnu.org/software/autoconf || autoconf
  • GNU Automake — Инструмент для автоматического создания make-файлов.
https://www.gnu.org/software/automake || automake
  • GNU Libtool — Скрипт поддержки общих библиотек.
https://www.gnu.org/software/libtool || libtool

Другие программы

Многие другие необязательные программы GNU доступны в официальных репозиториях:

  • GNOME — среда рабочего стола
  • GIMP — редактор изображений
  • GTK — инструментарий для разработки графических приложений
  • Gnumeric — табличный процессор
  • GNU Parted — инструмент для разметки дисков
  • GNU Screen — терминальный мультиплексор
  • GNU nano — консольный текстовый редактор
  • GNU Emacs — расширяемый, настраиваемый, самодокументирующийся текстовый редактор
  • GnuPG — реализация OpenPGP
  • GNU Octave — научный язык программирования
  • GNU Readline — библиотека для взаимодействия с командной строкой
