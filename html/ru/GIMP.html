GIMP (Русский)

Appearance
From ArchWiki

Ссылки по теме

  • GIMP/CMYK support
  • List of applications/Multimedia#Raster graphics editors

GIMP это мощная программа для редактирования растровых изображений, широко используемая для изменения фотографий, создания композиций и общего графического дизайна. Она может использоваться как простая программа для рисования, программа для редактирования фотографий экспертного качества, онлайн система пакетной обработки, рендерер для массового производства, конвертер форматов изображений и т. д.

Установка

Установите пакет gimp.

Советы и рекомендации

Подпись

Чтобы подписать изображение, выполните следующие действия после ввода желаемого текста:

  1. Нажмите Layer и Text to Path
  2. Нажмите Select и From Path
  3. Нажмите Select и Invert
  4. Нажмите Edit и Stroke Path

Нарисовать круг

Чтобы нарисовать круг в GIMP сделайте следующее:

  1. Выберите Ellipse tool и зажмите Shift
  2. Нажмите Edit и Fill with Color
  3. Нажмите Select и Shrink
  4. Нажмите Delete
Примечание Grow и Border даюь один результат.

Как Photoshop

Поскольку GIMP обладает широкими возможностями настройки, можно изменить сочетания клавиш и даже пользовательский интерфейс, чтобы он был более знаком тем, кто привык к программе редактирования растровых изображений Photoshop.

В частности GimpPs цель которого, сделать GIMP максимально схожим с Photoshop, который вы можете установить поверх GIMP.

Также, если вам просто нужны привязки клавиш, соответствующие строки включаются в файл menurc темы, который затем можно добавить в локальный файл ~/.config/GIMP/2.10/menurc

Существует также проект PhotoGIMP, целью которого является сделать то же самое.

Плагины

GIMP имеет обширную систему плагинов, а также Scheme интерпретатор, который можно использовать для записи Script-Fu скриптов.

Большинство плагинов находятся в официфльном репозитории (для примера, gimp-plugin-gmic) и AUR; некоторые, но не все, перечислены как необязательные зависимости пакета gimp.

Note Плагин Python-Fu недоступен в версии GIMP, распространяемой через официальные репозитории, поскольку для него требуется python2AUR, срок эксплуатации которого истек в 2020 году. Для восстановления функциональности можно использовать python2-gimpAUR

Ручная установка

Если плагины не распространяются через репозитории, их можно скомпилировать и установить с помощью исполняемого файла gimptool.

Чтобы установить плагин из исходного кода: 

$ gimptool --install source.c

Чтобы установить предварительно скомпилированный плагин: 

$ gimptool --install-bin executable

Чтобы установить скрипт Script-Fu: 

$ gimptool --install-script script.scm

Дополнительные варианты можно найти в выводе gimptool --help.

Создание пакетов

Пакеты плагинов должны добавлять свои файлы в каталог с их именем внутри. /usr/lib/gimp/2.0/plug-ins/.

Решение проблем

Зелёный текст

Добавьте следующее в ~/.config/GIMP/2.10/fonts.conf, если вы видите зеленый оттенок вокруг букв при включенном сглаживании: 

~/.config/GIMP/2.10/fonts.conf
<?xml version='1.0'?>
<!DOCTYPE fontconfig SYSTEM 'fonts.dtd'>
<fontconfig>
  <match target="font">
    <edit name="rgba" mode="assign">
      <const>none</const>
    </edit>
  </match>
</fontconfig>

Скрыть Noto

Добавьте следующее в ~/.config/GIMP/2.10/fonts.conf если у вас установлены noto-fonts и вы хотите удалить их из списка шрифтов GIMP: 

~/.config/GIMP/2.10/fonts.conf
<?xml version='1.0'?>
<!DOCTYPE fontconfig SYSTEM 'fonts.dtd'>
<fontconfig>
  <rejectfont>
    <glob>/usr/share/fonts/noto</glob>
  </rejectfont>
</fontconfig>

Проверите fonts-conf(5) и Font configuration для детальной информации.

PDF файлы

GIMP тредует библиотеку poppler-glib для открытия PDF файлы или он сообщит, что они не распознаны.

Поскольку GIMP растрирует PDF-файлы с самого начала, он не будет использовать встроенные возможности PDF при их редактировании. Для лучшего редактирования PDF файлов следует использовать программы на подобие (LibreOffice Draw).
