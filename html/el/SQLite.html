SQLite (Ελληνικά)

  • English
  • 日本語
  • 中文（简体）
From ArchWiki

Από την αρχική σελίδα του project:

Το SQLite είναι μία βιβλιοθήκη λογισμικού που ενσωματώνει μια αυτοτελή μηχανή βάσης δεδομένων SQL. Το SQLite είναι η πιο ευρέως χρησιμοποιούμενη μηχανή βάσεων δεδομένων SQL στον κόσμο. Ο πηγαίος κώδικας του SQLite είναι σε δημόσια διάθεση.

Εγκατάσταση

Εγκατάσταση του πακέτου sqlite.

Σχετικά πακέτα είναι:

  • sqlite-doc – τα περισσότερα στατικά αρχεία HTML που αποτελούν την ιστοσελίδα και περιλαμβάνουν όλη την σύνταξη SQL, τις προδιαγραφές της διεπαφής C/C++ και λοιπή τεκμηρίωση [1]
  • sqlite-analyzer – Το sqlite3_analyzer είναι ένα βοηθητικό πρόγραμμα γραμμής εντολών που μετράει και εμφανίζει πόσο αποτελεσματικά χρησιμοποιείται ο χώρος από τους εκάστοτε πίνακες ενός αρχείου βάσης δεδομένων SQLite [2]
  • sqlite-tcl – η διεπαφή Tcl στην βιβλιοθήκη SQLite [3]
  • php-sqlite – το άρθρωμα sqlite3 για PHP (μην ξεχάσετε να το ενεργοποιήσετε στο /etc/php/php.ini)
  • ruby-sqlite3 – Ruby bindings για την ενσωματωμένη βάση δεδομένων SQLite3
  • gambas3-gb-db-sqlite3 – τμήμα πρόσβασης βάσης δεδομένων Gambas3 Sqlite3

Χρησιμοποιώντας το κέλυφος γραμμής εντολών

Η βιβλιοθήκη SQLite περιέχει μία απλή λειτουργία γραμμής εντολών που ονομάζεται sqlite3 και επιτρέπει στον χρήστη την χειροκίνητη εισαγωγή και εκτέλεση εντολών SQL στην βάση δεδομένων SQLite.

Δημιουργία βάσης δεδομένων 

$ sqlite3 databasename

Δημιουργία πίνακα 

sqlite> create table tblone(one varchar(10), two smallint);

Εισαγωγή δεδομένων 

sqlite> insert into tblone values('helloworld',20);
sqlite> insert into tblone values('archlinux', 30);

Αναζήτηση βάσης δεδομένων 

sqlite> select * from tblone;
helloworld|20
archlinux|30
