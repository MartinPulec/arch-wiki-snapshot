rTorrent (Français)

Appearance
From ArchWiki

rTorrent est un client BitTorrent écrit en C++ et utilisant la bibliothèque logicielle libTorrent (qui ne doit pas être confondue avec libtorrent-rasterbar). Celle-ci est développée conjointement au client. rTorrent fournit un environnement en mode texte grâce à la bibliothèque ncurses. Combiné avec un multiplexeur de terminal, comme GNU Screen ou Tmux et Secure Shell, il peut s'avérer pratique pour une utilisation à distance.

Installation

Installez le paquet rtorrent.

Paquets non-officiels

  • rtorrent-gitAUR - Branche principale du dépôt Git du paquet.
  • rtorrent-psAUR - Version contenant l'ensemble des correctifs rtorrent-ps.
  • rtorrent-pyro-gitAUR - Autre branche du dépôt Git du paquet avec l'ensemble des correctifs rtorrent-ps.
  • rtorrent-vi-colorAUR[broken link: package not found] - Version avec des raccourcis claviers inspirés de Vi.

Raccourcis clavier

La navigation au sein de rTorrent se fait exclusivement à l'aide de raccourcis clavier. Un résumé de ceux-ci est disponible ci-dessous. Un guide plus complet est disponible sur le wiki de rTorrent (voir: Guide utilisateur de rTorrent).

Note Appuyer deux fois successivement sur Ctrl-q arrêtera aussitôt rTorrent sans avertir les trackers connectés.
Cmd Action
Ctrl-q Quitte l'application.
Ctrl-s Démarre le téléchargement. Lance la vérification d'intégrité sauf si celle-ci a déjà été effectuée.
Ctrl-d Arrête un téléchargement en cours ou supprime un téléchargement arrêté.
Ctrl-k Arrête et ferme les fichiers d'un téléchargement en cours.
Ctrl-r Commence la vérification d'intégrité. Démarre le téléchargement si le fichier est introuvable.
Ctrl-o Spécifie le répertoire de téléchargement pour un torrent ajouté mais pas encore démarré.
Left Retour au menu précédent.
Right Aller au menu suivant.
Backspace Ajoute et démarre le téléchargement du torrent spécifié.
Return Ajoute et mais ne démarre pas le téléchargement du torrent spécifié.
a|s|d Augmente la limite générale d'envoi de 1|5|50 Ko/s.
A|S|D Augmente la limite générale de téléchargement de 1|5|50 Ko/s.
z|x|c Diminue la limite générale d'envoi de 1|5|50 Ko/s.
Z|X|C Diminue la limite générale de téléchargement de 1|5|50 Ko/s.

Raccourcis clavier redondants

Ctrl-s est souvent utilisé pour contrôler l'arrêt de l'affichage au sein d'un terminal tandis que Ctrl-q est utilisé pour le démarrer. Ces raccourcis clavier peuvent interférer avec rTorrent. Il est possible de vérifier si ces derniers sont configurés : 

$ stty -a
...
swtch = <undef>; start = ^Q; stop = ^S; susp = ^Z; rprnt = ^R; werase = ^W; lnext = ^V;
...

Pour supprimer ces raccourcis, il faut changer les options du terminal afin de supprimer ces définitions (ici stop et start): 

# stty stop undef
# stty start undef

Pour les supprimer automatiquement au démarrage, il est possible d'ajouter les deux commandes au fichier ~/.bashrc.
