Le support d'installation officiel d'Arch Linux prend en charge diverses fonctionnalités d'Accessibilité :

  • la synthèse vocale est fournie par le paquet espeakup
  • la prise en charge du terminal en braille est fourni par le paquet brltty

Ce document décrit comment installer Arch Linux en utilisant ces fonctionnalités.

Note: Cette page ne décrit que les étapes qui sont différentes du guide d'installation.

Pré-installation

Démarrer l'environnement «live»

Note: Les chargeurs d'amorçage du support d'installation attendent 15 secondes avant de lancer l'élément par défaut du menu

Quand le support d'installation commence à démarrer, appuyez sur Bas suivi de Entrée pour démarrer avec la synthèse vocale activée.

Les écrans braille branchés en USB devraient être détectés automatiquement par udev.

Cartes son multiples

Si votre ordinateur dispose de plusieurs cartes son, vous entendrez le message suivant : Please select your sound card for speech output.

Quand un bip retentit sur la sortie audio que vous souhaitez utiliser, appuyez sur Entrée pour la sélectionner.

Changer la langue de la synthèse vocale

Pour modifier la langue de espeak-ng utilisée par espeakup.service, éditez l'unité de façon à ce que le code de la langue soit ajouté à default_voice= dans la partie Environment=.

Il est également possible de changer la voix de espeak-ng en ajoutant +variant au code de langue. Consultez espeakup(8) et espeak-ng(1) pour plus d'informations.

Installation

Installation des paquets essentiels

Pour la prise en charge de la synthèse vocale dans le système installé, vous avez besoin de espeakup et alsa-utils. Si vous utilisez un écran braille, installez le paquet brltty.

Ajoutez les paquets souhaités à la commande pacstrap(8) lors de l'installation : 

# pacstrap -K /mnt base linux linux-firmware espeakup alsa-utils

Configurer le système

Carte son

Si des #Cartes son multiples ont été détectées, copiez le fichier /etc/asound.conf, qui a été généré dans le support d'installation : 

# cp /etc/asound.conf /mnt/etc/

Activer les services

Pour bénéficier d'une assistance vocale après le démarrage du système installé, activez espeakup.service. Consultez également #Changer la langue de la synthèse vocale.

Redémarrage

Après redémarrage du système nouvellement installé, la synthèse vocale devrait commencer automatiquement.
