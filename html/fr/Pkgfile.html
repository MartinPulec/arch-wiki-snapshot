pkgfile est un outil pour rechercher des fichiers dans les paquets des dépôts officiels.

Tip: pacman a une fonctionnalité similaire intégrée (en)

Installation

Installez le paquet pkgfile. Vous pouvez également installer la version de développement avec le paquet pkgfile-gitAUR.

La base de données pkgfile peut alors être synchronisée avec : 

# pkgfile -u

Utilisation

Pour rechercher un paquet qui contient le fichier makepkg : 

$ pkgfile makepkg
core/pacman

Pour lister tous les fichiers fournis par archlinux-keyring: 

$ pkgfile -l archlinux-keyring
core/archlinux-keyring usr/
core/archlinux-keyring usr/share/
core/archlinux-keyring usr/share/pacman/
core/archlinux-keyring usr/share/pacman/keyrings/
core/archlinux-keyring usr/share/pacman/keyrings/archlinux-revoked
core/archlinux-keyring usr/share/pacman/keyrings/archlinux-trusted
core/archlinux-keyring usr/share/pacman/keyrings/archlinux.gpg

Cette dernière est comparable à pacman -Ql (Consulter la sectioninterrogation de la base de données des paquets de pacman), sauf qu'il s'applique aux paquets distants.

Commande introuvable

See Bash#Command not found(en), Zsh#pkgfile "command not found" handler(en) and Fish#The "command not found" hook(en).

Mises à jour automatiques

pkgfile est livré avec un service systemd et timer pour synchroniser automatiquement la base de données pkgfile. Pour activer les mises à jour automatiques, activez pkgfile-update.timer.

Par défaut, pkgfile sera mis à jour quotidiennement. Pour modifier ce calendrier, modifiez les unités.
