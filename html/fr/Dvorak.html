Selon Wikipedia :

Dvorak est une disposition de clavier pour l'anglais brevetée en 1936 par August Dvorak et son beau-frère, William Dealey, comme une alternative plus rapide et plus ergonomique à la disposition QWERTY (la disposition de clavier standard de facto). Les partisans de la disposition Dvorak affirment qu'elle nécessite moins de mouvements des doigts et que, par conséquent, elle réduit les erreurs, augmente la vitesse de frappe, réduit les microtraumatismes répétés ou est simplement plus confortable que la disposition QWERTY.

Configuration de la disposition Dvorak

Consultez Configuration du clavier en console ou Configuration du clavier dans Xorg pour les détails de configuration.

Pour le terminal virtuel, dvorak et le clavier régional sont combinés en un seul keymap. Mais Xorg liste dvorak comme variante de votre clavier régional.

Les claviers Dvorak us pour le terminal virtuel sont les suivants :

  • dvorak, Standard
  • dvorak-l, Dvorak pour gauchers
  • dvorak-r, Dvorak pour droitiers
  • dvorak-programmer, Dvorak du programmeur

Les us Dvorak pour Xorg sont les suivants :

  • dvorak, Standard
  • dvorak-l, Dvorak pour gauchers
  • dvorak-r, Dvorak pour droitier
  • dvp, Dvorak pour programmeur
  • dvorak-intl, Dvorak international
  • dvorak-classic, Dvorak international
  • {ic|dvorak-alt-intl}}, Dvorak international
Note: Pour la console, ce sont des keymaps autonomes, mais pour Xorg, ce sont des variantes de la disposition us, vous devez les passer à la variable XkbVariant. Consultez Xorg (Français)/Keyboard configuration (Français)#Configurer la disposition du clavier pour une explication.

Pour les utilisateurs internationaux

Français

La variante française s'appelle Bépo.

Suédois

Les Suédois désireux d'essayer dvorak peuvent trouver la "version" suédoise, appelée svorak, sur aoeu.info ! Pour convertir en svorak sous X, vous n'avez pas besoin de télécharger de fichiers supplémentaires depuis www.aoeu.info.

Espagnol

Dans la console, spécifiez dvorak-es au lieu de dvorak pour utiliser la variante espagnole de dvorak.

Dans Xorg, spécifiez es comme XkbLayout et dvorak comme XkbVariant.

Royaume-Uni

En console, spécifiez dvorak-ukp (disponible à partir de dvorak-ukpAUR) au lieu de dvorak pour utiliser la variante dvorak du Royaume-Uni avec la ponctuation ISO/IEC 9995-1.

Dans Xorg, spécifiez gb comme XkbLayout et dvorakukp comme XkbVariant.

Raccourcis spécifique aux programmes

Il s'agit de quelques suggestions simples et raisonnables pour la réaffectation dans les programmes centrés sur le clavier. Comme les touches de Dvorak penchent davantage vers la droite du clavier, la ligne d'accueil utilisée ici est htns.

Vim

~/.vimrc
noremap h <gauche>
noremap t <bas>
noremap n <haut>   
noremap s <droite>

noremap l n
noremap L N
Note: Il s'agit de la commande n search repeat et n'a rien à voir avec la touche n. Consultez :help n.

Consultez également le Wiki de Vim, si vous souhaitez une approche plus extrême.

Mutt

~/.config/mutt/muttrc
bind index t next-entry
bind index n previous-entry
bind index s display-message

bind index G last-entry
bind index gg first-entry

bind pager,attach h exit
bind pager t next-line
bind pager n previous-line
bind pager s view-attachments

bind browser h goto-parent
bind pager,browser gg top-page
bind pager,browser G bottom-page

Less

~/.lesskey
t forw-line
n back-line
l repeat-search 
L reverse-search

Pour appliquer les paramètres, vous devez exécuter lesskey.

Zathura

~/.config/zathura/zathurarc
map h scroll left
map t scroll down
map n scroll up
map s scroll right

map l search forward
map L search backward

Qutebrowser

~/.config/qutebrowser/config.py
config.bind('h', 'scroll left')
config.bind('t', 'scroll down')
config.bind('n', 'scroll up')
config.bind('s', 'scroll right')

config.bind('H', 'back')
config.bind('T', 'tab-prev')
config.bind('N', 'tab-next')
config.bind('S', 'forward')

config.bind('l', 'search-next')
config.bind('L', 'search-prev')

config.bind('e', 'hint all')
