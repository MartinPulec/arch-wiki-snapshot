Bottles (Français)

  • English
  • 日本語
  • Русский
From ArchWiki

Bottles est un gestionnaire de préfixes Wine écrit en Python et utilisant le framework GTK. Il peut être utilisé pour créer et gérer des préfixes, automatiser l'installation de lanceurs Wine, de dépendances et d'applications Windows. Il peut aussi surcharger les fichiers DLL Windows à l'intérieur d'un préfixe et de gérer les variables d'environnement des sessions Wine.

Bottles permet d'exécuter des applications et des jeux Windows natifs avec, dans la plupart des cas, des performances quasi-natives.

Note: Les développeurs de Bottles recommandent très fortement d'installer l'application via Flatpak afin de permettre à l'application de s'exécuter dans une sandbox

Installation

Bottles peut être installée via Flatpak avec la commande suivante : 

$ flatpak install bottles

Bottles est aussi disponible sous la forme d'un paquet AUR, nommé bottles-gitAUR.

Utilisation

Bottles dispose d'une documentation détaillée, destinée aux utilisateurs.
