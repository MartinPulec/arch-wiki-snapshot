Package Maintainer (Français)

Appearance
From ArchWiki

Mainteneurs de paquet (anciennement appelé utilisateur de confiance) est une rôle du personnel d'Arch Linux officiel. Les mainteneurs de paquets remplissent les taches :

  • Maintenir les paquets dans le dépôt officiel (à part celui de core).
  • Maintenir, la gestion, et surveiller les opérations de l'AUR.

Le terme générique "mainteneur de paquets" est aussi utilisé pour décrire n'importe qui maintenant un paquet, de n'importe qu'elle dépôt, comme décrit dans terminologie Arch#Mainteneur de paquet

Comment devenir un mainteneur de paquet

Le minimum requis pour devenir un mainteneur de paquet est le suivant :

  • connaître les bases du shell scripting
  • maintenir quelque paquet dans l'AUR avec un PKGBUILDs soigné et de forte qualité
  • engagement communautaire basique (liste de diffusions, forum, IRC)
  • maîtrise avec la recherche web
  • une idée générale du type de paquet que vous voulez maintenir(globalement, pourquoi vous voulez devenir un mainteneur de paquet)

Même si vous pouvez devenir un mainteneur de paquet en remplissant simplement le minimum des attendus, les personnes vous jugeant durant la procédure de vote standard peuvent espérer plus venant de vous. Comme par exemple :

  • Engagement dans les bug tracker (reportage, information)
  • Des Patches pour les projets Arch Linux
  • Engagement dans quelque projets open-source(même si ces les vôtres)

Si vous voulez toujours devenir un mainteneur de paquets après avoir lus ses lignes, le premier pas est de trouver deux mainteneur de paquets qui sont d'accord de vous sponsoriser. Une fois sponsorisé, vous devrez rédiger une candidature plein d'esprit signée avec vôtre clé OpenPGP pour la liste de diffusion du aur-general.

Note Si un Mainteneur de paquet que vous contactez refuse de sponsoriser votre demande, vous devez en informer cela si vous demandez une sponsorisation auprès d'un autre Maintenance de Package.

Pour plus d'information, regardez le Mainteneur de paquet Bylaws et les règles des mainteneurs de paquets.
