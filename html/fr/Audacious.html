Audacious est un lecteur audio basé sur GTK+. Il vise à pourvoir une bonne qualité audio et prends en charge une grande variété de codecs audio tout en restant assez léger. Il est facilement extensible via l'ajout de plugins.

Installation

Vous pouvez installer le paquet audacious avec Pacman.

Configuration

Plugins

AMIDI-Plug (MIDI Player)

Pour jouer les fichiers MIDI et RMI, il est nécessaire d'installer le paquet fluidsynth,ainsi que les bibliothèques de sons (sound fonts) freepats-general-midi et soundfont-fluid. Aucune configuration particulière n'est requise par fluidsynth package, mais pour des fonctionnalités additionnelles, consultez la page FluidSynth (anglais)

À la suite de cela, le plugin sera activé. Et vous pourrez ajouter les bibliothèques de son dans le panneau de configuration configuration (Fichier, Paramètres ..., Panneaux Plugins, Onglet Input, sélectionnez AMIDI-Plug (MIDI Player)) et ajoutez les bibliothèques de son ((Extension .sf2 situées dans /usr/share/soundfonts/) dans la boite de dialogue, une après l'autre.

Trucs et astuces

Audtool

Audacious est fourni avec un puissant outil de management appelé Audtool. Il peut être utilisé pour récupérer des informations ou pour contrôler le lecteur.

Par exemple, pour récupérer les informations à propos du titre ou de l'artiste, utilisez la commande suivante : 

audtool current-song
audtool curretn-song-tuple-data artist

Il existe de nombreuses autres fonctions, pour contrôler le playback, manipuler la playlist ou la fenêtre principale, etc. Pour la liste complète des options voir audtool --help et audtool(1).

Winamp

Audacious propose actuellement deux interfaces:

  • Une interface classique Winamp
  • Une interface Qt

Il est possible de basculer de l'une à l'autre de ces interfaces dans Audacious.

Ajouter des thèmes Winamp pour Audacious est très simple. Il suffit de copier le fichier de thème (.zip, .tgz., .tar.gz, ou .tar.bz2) soit dans ~/.local/share/audacious/Skins, ce qui n'affectera que l'utilisateur, soit dans /usr/share/audacious/Skins, ce qui affectera tous les utilisateurs, puis de chercher et sélectionner le thème voulu dans Préférences -> Thèmes. Il est également possible de déposer le fichier thème directement au sein de la liste des thèmes disponibles.
