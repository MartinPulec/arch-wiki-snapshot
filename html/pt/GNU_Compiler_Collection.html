O GNU Compiler Collection (GCC) é parte do conjunto de ferramentas do GNU e inclui front-ends para C e C++.

Instalação

Instale o pacote gcc.

Outros front-ends disponíveis são:

  • gcc-ada para Ada
  • gcc-fortran para Fortran
  • gcc-go para Go
  • gcc-objc para Objective-C
