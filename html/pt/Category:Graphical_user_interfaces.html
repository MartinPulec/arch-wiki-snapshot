Essa categoria destina-se aos artigos gerais de interface gráfica do usuário (GUI), aplicativos GUI pertencem a Category:Applications (Português).

Ambiente de desktop
Fornece uma interface gráfica de usuário (GUI) completa para um sistema agrupando uma variedade de componentes.
Gerenciador de janela
É o software de sistema que controla o posicionamento e a aparência de janelas dentro de um sistema de janelas em uma GUI.
Gerenciador de exibição
É um gerenciador de login gráfico que inicia uma sessão em um servidor de exibição.

Para drivers de placa de vídeo, veja Category:Graphics (Português).
