A mídia de instalação oficial do Arch Linux oferece suporte a vários recursos de Acessibilidade:

  • fala é fornecida pelo pacote espeakup
  • suporte a terminal braille é gerenciado pelo pacote brltty

Este documento descreve como instalar o Arch Linux usando esses recursos.

Nota: Esta página descreve apenas as etapas que são diferentes do Guia de instalação.

Pré-instalação

Inicializar o ambiente live

Nota: Os gerenciadores de boot na mídia de instalação têm um tempo limite de 15 segundos antes de iniciarem o item de menu padrão.

Quando a mídia de instalação começar a inicializar, pressione Seta para baixo seguido por Enter para habilitar a inicialização com fala habilitada.

Dispositivos braille USB devem ser detectados automaticamente via udev.

Várias placas de som

Se o seu computador tiver várias placas de som, você ouvirá a seguinte mensagem: Please select your sound card for speech output. Significa "Por favor, selecione sua palca de som para a fala."

Ao ouvir um bipe na saída que gostaria de usar, pressione Enter para selecionar a placa.

Alterar o idioma da fala

Para alterar a voz/idioma do espeak-ng usado por espeakup.service, edite a unit de forma que o código de idioma seja anexado ao default_voice= na diretiva Environment=.

Você também pode alterar a variante de voz do espeak-ng anexando +variante ao código de idioma. Veja espeakup(8) e espeak-ng(1) para mais informações.

Instalar

Instalar os pacotes essenciais

Para suporte de fala no sistema instalado, você precisa de espeakup e alsa-utils. Se você usa um dispositivo braille, instale o pacote brltty.

Anexe os pacotes necessários à chamada pacstrap(8) ao instalar: 

# pacstrap /mnt base linux linux-firmware espeakup alsa-utils

Configurar o sistema

Placa de som

Se #Várias placas de som foram detectadas, copie o arquivo /etc/asound.conf, o qual foi gerado na mídia de instalação: 

# cp /etc/asound.conf /mnt/etc/

Habilitar os serviços

Para ter suporte de voz após inicializar no sistema instalado, você precisa habilitar espeakup.service. Veja também #Alterar o idioma da fala.

Reiniciar

Depois de inicializar no sistema recém-instalado, ele deve começar a falar automaticamente.
