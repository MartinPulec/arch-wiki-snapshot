rbenv (Simple Ruby Version Management) é uma ferramenta que permite alternar facilmente entre várias versões do Ruby.

Outra ferramenta a ser utilizada para o mesmo fim é RVM.

Instalação

Instalar o pacote rbenv

Plugins

O rbenv pode ser estendido por meio de um sistema de plugins, e o wiki do rbenv inclui uma lista de plugins úteis. O plug-in ruby-build é especialmente útil, pois permite instalar versões do Ruby com o rbenv install comando. Você pode instalar o ruby-build do AUR.
