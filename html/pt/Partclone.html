Partclone, assim como o bem conhecido Partimage, pode ser usado para copiar e restaurar uma partição enquando considerando apenas os blocos usados.

Instalação

Instale o pacote partclone.

Usando Partclone com uma partição formatada em ext4

Sem compressão

Para copiar sem compressão: 

$ partclone.ext4 -c -s /dev/sda1 -o ~/image_sda1.pcl

Para restaurá-la: 

$ partclone.ext4 -r -s ~/image_sda1.pcl -o /dev/sda1

Com compressão

Para copiar compressão: 

$ partclone.ext4 -c -s /dev/sda1 | gzip -c > ~/image_sda1.pcl.gz
Nota: Para compressão máxima, use gzip -c9

Para restaurá-la: 

zcat ~/image_sda1.pcl.gz | partclone.ext4 -r -o /dev/sda1
