From ArchWiki

O artigo principal para essa categoria é Laptop.

Esta categoria contém artigos sobre como instalar e configurar o Arch Linux em laptops. As páginas específicas do modelo devem ser categorizadas na subcategoria apropriada do fabricante para evitar confusão.

Nota:
  • Para obter instruções gerais de instalação do Arch Linux, você deve sempre seguir o guia de instalação e usar as respectivas páginas de laptop para conhecer as especificidades de hardware. Para instruções pós-instalação, Recomendações gerais e Laptop são páginas meta para orientar o conteúdo relevante a ser seguido em paralelo a qualquer configuração específica de hardware que os artigos desta categoria possam conter.
  • Se você quiser adicionar informações, considere primeiro onde elas se encaixam melhor. Em particular, não duplique as instruções existentes de outros artigos do ArchWiki para artigos de laptop. Em vez disso, detalhe a configuração específica do hardware e trabalhe com a vinculação de instruções existentes ou melhore-as se sua contribuição for aplicável a outro hardware.
  • Se você deseja incluir a saída de lspci, certifique-se de mencionar o número do modelo específico e adicione as informações no fim do artigo.
  • Se você quiser criar um novo artigo, veja esses dois exemplos:
    • Um exemplo de um artigo de laptop com um mínimo de duplicação de conteúdo e um foco no hardware específico é ASUS N550JV.
    • Outro exemplo é o Acer C720 Chromebook, que detalha instruções específicas de hardware e está relacionado a dispositivos Chrome OS como uma meta-página separada, porque a própria categoria de dispositivo requer etapas de instalação muito específicas.
Por favor, considere usar estruturas de conteúdo similar para manter o foco.
Dica: Para uma lista mais extensa de artigos (em inglês), recomenda-se ver a página original Category:Laptops.
