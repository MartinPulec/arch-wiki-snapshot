echinus es un gestor de ventanas flotantes y de mosaico simple y liviano para X11. Comenzó como una bifurcación dwm con una configuración más sencilla, echinus se convirtió en un gestor de ventanas de reparentalización con todas las funciones con soporte EWMH.

A diferencia de dwm, echinus no necesita volver a compilarse después de realizar cambios en la configuración. Es compatible con Xft (freetype) y tiene la opción de barras de título configurables.


Instalación

Instale echinusAUR. También puede instalar ouricoAUR, una barra de tareas EWMH liviana, diseñada originalmente para echinus (también en AUR), y dmenu.

Después de instalar con éxito, copie todos los archivos de /etc/xdg/echinus a ~/.echinus (para el usuario).

Configuración

echinus se configura en un archivo de texto simple, en formato Xresources: ~/.echinus/echinusrc. Los detalles de todas las opciones de configuración se encuentran en /usr/share/doc/echinus/README. Una sección de un archivo de configuración de muestra tal que así: 

Echinus*selected.border: #404040
Echinus*selected.button: #d3d7cf
Echinus*selected.bg: #262626
Echinus*selected.fg: #d3d7cf

Reglas

Las reglas se pueden configurar para generar aplicaciones en etiquetas específicas. La siguiente regla, por ejemplo, abriría Firefox en la etiqueta "web": 

Echinus*rule0: firefox.* web 0 1

Abrir aplicaciones en un terminal requiere que establezcas explícitamente la etiqueta -title al generarlas para que echinus pueda administrarlas: 

Echinus*spawn0: CA + m = urxvtc -title mutt -e mutt

De manera similar, al generar dmenu, deberá declarar las propiedades requeridas, así: 

Echinus*spawn1: Menu = dmenu_run -fn "-*-dina-medium-r-*-*-*-100-*-*-*-*-*-*" -nb "#1A1A1A" -nf "#696969" -sb "#1A1A1A" -sf "#D3D7Cf"

Iniciar echinus

Para iniciar echinus con startx o el gestor de inicio de sesión SLiM, simplemente añada lo siguiente a ~/.xinitrc: 

exec echinus

Utilizar echinus

Después de realizar cambios en echinusrc, puede volver a cargar la configuración sin volver a compilar reiniciando echinus, con Alt + Mayús + q. Esta combinación de teclas, y cualquier otra, se puede personalizar para adaptarse a sus preferencias o memoria muscular.

Más detalles para manipular ventanas están en la página de manual y en README.

Paneles y paginadores

echinus es compatible con algunas partes de EWMH; se sabe que las siguientes funcionan:

  • tint2
  • fbpanel
  • ipager[enlace roto 2025-03-15]
  • ouricoAUR
