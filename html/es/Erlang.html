Erlang es un lenguaje de programación con las cualidades específicas de datos inmutables y computación distribuida.

Instalación

Instale el paquete erlang.

Utilización

Para obtener más detalles, véase la documentación de Erlang.

Para activar la consola, ejecute la orden erl

En la cual puede ingresar órdenes como: 

   Erlang/OTP 21 [erts-10.1] [source] [64-bit] [smp:4:4] [ds:4:4:10] [async-threads:1] [hipe]
   Eshell V10.1  (abort with ^G)
   1> 1 + 2.
   3
   2> (2 + 3) * 4 / 5.
   4.0
   3>

Consejos y trucos

Modo Emacs

Para configurar el modo Emacs incluido, erlang-mode, véase la documentación (la ruta OTP es /usr/lib/erlang).
