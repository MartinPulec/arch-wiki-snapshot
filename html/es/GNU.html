De Wikipedia:

GNU es un sistema operativo y una extensa colección de software. GNU se compone íntegramente de software libre, la mayor parte del cual está licenciado bajo la Licencia Pública General (GPL) del Proyecto GNU. GNU es el acrónimo recursivo de "¡GNU no es Unix!".

Dado que el núcleo GNU, Hurd, no está listo para producción [1], GNU se suele usar con el núcleo Linux. Arch Linux es una de estas distribuciones GNU/Linux, que utiliza software GNU como el shell Bash, los coreutils de GNU, la cadena de herramientas GNU y numerosas otras utilidades y bibliotecas. Esta página no pretende enumerar todos los [2] paquetes GNU, y solo destaca algunos.

Texinfo

El software GNU se documenta utilizando la sintaxis de composición tipográfica Texinfo. Puede ver documentos Info utilizando el programa info, proporcionado por el paquete texinfo.

Si bien la mayoría del software GNU también proporciona man pages, los documentos Info suelen ser más completos. Para ver un documento de información, simplemente introduzca: 

$ info page_name

Sistema base

  • Bash — Es un shell compatible con sh que incorpora funciones útiles del shell Korn (ksh) y el shell C (csh).
https://www.gnu.org/software/bash/ || bash
  • Coreutils — Coreutils proporciona las utilidades básicas de manipulación de archivos, shell y texto del sistema operativo GNU.
https://www.gnu.org/software/coreutils/ || coreutils
  • GRUB — GRUB es el boot loader del proyecto GNU.
https://www.gnu.org/software/grub/ || grub
  • gzip — gzip es tanto un formato de archivo como una aplicación de software para compresión y descompresión.
https://www.gnu.org/software/gzip/ || gzip
  • tar — Permite crear o descomprimir archivos tar, así como realizar otras manipulaciones.
https://www.gnu.org/software/tar/ || tar

Cadena de herramientas

La mayoría de las herramientas de la GNU toolchain son ​​dependencias del paquete base-devel, excepto glibc (requerida por base) y GDB.

  • Binutils — Un conjunto de programas para ensamblar y manipular archivos binarios y de objeto. Incluye ld.
https://www.gnu.org/software/binutils/ || binutils
  • Bison — El generador de analizadores sintácticos de propósito general de GNU.
https://www.gnu.org/software/bison/bison.html || bison
  • GCC — La colección de compiladores de GNU: interfaces para C y C++.
https://gcc.gnu.org/ || gcc
  • GDB — El depurador de GNU.
https://www.gnu.org/software/gdb/ || gdb
  • glibc — Implementación de GNU de la biblioteca C.
https://www.gnu.org/software/libc/ || glibc (requerido por base)
  • m4 — El procesador de macros de GNU.
https://www.gnu.org/software/m4/ || m4
  • make — Utilidad de creación de GNU para el mantenimiento de grupos de programas.
https://www.gnu.org/software/make || make

Sistema de compilación

De Wikipedia:

El sistema de compilación de GNU o GNU Build System, conocido también como Autotools, es un conjunto de herramientas producido por el proyecto GNU. Estas herramientas están diseñadas para ayudar a crear paquetes de código fuente portable a varios sistemas Unix.
  • Autoconf — Herramienta para configurar automáticamente el código fuente.
https://www.gnu.org/software/autoconf || autoconf
  • Automake — Herramienta para crear automáticamente Makefiles.
https://www.gnu.org/software/automake || automake
  • Libtool — Un script genérico de soporte para bibliotecas.
https://www.gnu.org/software/libtool || libtool
