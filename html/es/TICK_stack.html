La pila TICK (TICK stack) es una colección de tecnologías asociadas que consiste en Telegraf InfluxDB Chronograf y Kapacitor. Se combinan para ofrecer una plataforma para almacenar, capturar, supervisar y visualizar datos que estén en series de tiempo.

La pila TICK consta de las siguientes tecnologías:

  • #Telegraf – recopilación de datos secuenciales vinculados de una variedad de fuentes, incluidos dispositivos IoT.
  • #InfluxDB – almacenamiento de base de datos eficiente y de alto rendimiento para manejar grandes volúmenes de datos de series temporales.
  • #Chronograf – visualización en tiempo real de datos InfluxDB.
  • #Kapacitor – supervisión y alertas basadas en vistas de datos de InfluxDB y anomalías contenidas en esas vistas.
Nota: Estos componentes no tienen porqué utilizarse todos juntos. También se pueden reemplazar con otras herramientas (por ejemplo, Grafana en lugar de Chronograf) o no utilizarse directamente (por ejemplo, Kapacitor).

Telegraf

Instalación

Instale el paquete telegrafAUR o telegraf-binAUR.

Configuración

El archivo de configuración principal se aloja en /etc/telegraf/telegraf.conf. Para complementos, configuración y personalización, véase upstream documentation.

También inicie y active telegraf.service.

InfluxDB

Instalación

Instale el paquete influxdb.

Configuración

El archivo de configuración principal se aloja en /etc/influxdb/influxdb.conf. Para la configuración, véase la documentación.

También inicie y active influxdb.service.

Chronograf

Instalación

Instale el paquete chronografAUR o chronograf-binAUR.

Configuración

El archivo de configuración principal se aloja en /etc/chronograf/chronograf.conf. Para la configuración, véase la documentación.

También inicie y active chronograf.service.

Kapacitor

Instalación

Instale el paquete kapacitorAUR o kapacitor-binAUR.

Configuración

El archivo de configuración principal se almacena en /etc/kapacitor/kapacitor.conf. Para la configuración, véase la documentación. También se pueden encontrar muchos ejemplos en el repositorio oficial de github.

También inicie y active kapacitor.service.
