< GNOME (Español)
Esta traducción de GNOME/Document viewer fue revisada el 2020-12-12. Si existen cambios puede actualizarla o avisar al equipo de traducción.

El visor de documentos está específicamente diseñado para admitir los siguientes formatos de archivo: PDF, PostScript, DjVu, tiff, dvi, XPS, soporte de SyncTex con gedit, libros de cómics (cbr, cbz, cb7 y cbt). Para obtener una lista completa de los formatos admitidos, véase Formatos de documentos compatibles.

El visor de documentos utiliza la biblioteca poppler como backend.

Nota: El visor de documentos era conocido anteriormente como Evince hasta que la aplicación recibió nuevos nombres descriptivos, uno para cada idioma admitido. El nombre Evince todavía se utiliza en muchos lugares, como el nombre del ejecutable, algunos nombres de paquetes, algunas entradas de escritorio y algunos esquemas GSettings.

Instalación

Instale el paquete evince, o evince-gitAUR para la versión de desarrollo. Evince instala gnome-desktop como dependencia.

Para una versión independiente instale evince-no-gnomeAUR[enlace roto: package not found] o la versión ligera evince-lightAUR[enlace roto: package not found] para tener soporte solamente de PDF.

Solución de problemas

La impresora no aparece

Actualice gtk3 a la versión 3.22.26+47+g3a1a7135a2-1 o superior. En versiones anteriores de GTK 3, los backends de la impresora GTK se incluyeron en un paquete separado. [1]

La ampliación es limitada

El aumento del tamaño de la memoria caché de la página de Evince le permite ampliar aún más, lo que es útil para documentos grandes. Por defecto, la configuración se establece en 50MiB. El aumento del tamaño del caché de la página obviamente aumenta el consumo de memoria de Evince cuando se amplía.

La siguiente orden aumenta el tamaño del caché de la página a un gigabyte: 

$ gsettings set org.gnome.Evince page-cache-size 'uint32 1000'

Los textos en PDF no se muestran correctamente

Intente establecer el parámetro override-restrictions a falso: 

$ gsettings set org.gnome.Evince override-restrictions false

La búsqueda inversa con SyncTeX no funciona

Compruebe que python-dbus está instalado. Tras esto Ctrl+click debería funcionar.

Soporte de libro de cómics WebP

Algunos archivos de libro de cómics (cbr, cbz etc.) utilizan imágenes WebP. Instale webp-pixbuf-loader para el soporte de libro de cómics WebP.

Consejos y trucos

Gestión de anotaciones

Evince v3.31.0 añade el atajo de teclado "s" para incluir anotaciones de texto y "Ctrl+H" para añadir una anotación de texto resaltado.

El autor predeterminado para las animaciones de texto de nota es igual al comentado en GECOS del usuario actual, para cambiar esto: 

usermod -c “Su nuevo nombre real completo” sunombredeusuario

Utilizar como visor de PDF predeterminado

Para establecer la asociación predeterminada para xdg-open, 

xdg-mime default org.gnome.Evince.desktop application/pdf

Otros abridores de recursos se pueden configurar de manera similar.
