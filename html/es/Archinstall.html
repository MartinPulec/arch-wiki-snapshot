archinstall es una librería auxiliar que automatiza la instalación de Arch Linux. Está empaquetada con diferentes instaladores pre-configurados, como un instalador "guiado".

Este documento no trata sobre el uso de archinstall como una librería de Python; revise la documentación oficial para eso.

Advertencia:
  • La manera recomendada de instalar Arch Linux continúa siendo seguir la guía de instalación.
  • archinstall almacena todas las contraseñas de usuario y de cifrado de discos (secundarios) en texto plano [1]
  • archinstall proporciona diferentes valores por defecto en comparación con el proceso de instalación regular. Cuando utilice un sistema instalado con archinstall, por favor, menciónelo en las solicitudes de soporte y proporcione el archivo /var/log/archinstall/install.log.

Ejecutando el instalador

Primero, adquiera y arranque el entorno live conforme a como se indica en la guía de pre-instalación. El paquete archinstall hace parte del medio vivo y puede ser ejecutado directamente: 

# archinstall

El instalador guiado le solicitará que introduzca datos en varios pasos, descritos en la documentación oficial[enlace roto 2024-01-13].

Advertencia: Dejar la contraseña root vacía deshabilita la cuenta del usuario root. Debido a que esto puede causar que se bloqueé a sí mismo fuera de su sistema, esto no se aconseja generalmente. Revise Sudo (Español)#Desactivar el acceso de superusuario para más información.
Nota: El instalador puede configurar interfaces cableadas en el sistema instalado utilizando systemd-networkd (Español) o puede copiar la configuración usada en la imagen ISO. Es decir, si usted configura una interfaz inalámbrica utilizando Iwd#iwctl, la configuración, incluyendo la contraseña de la red, será copiada al sistema instalado. La configuración de interfaces cableadas que estén presentes en la imagen ISO también será copiada.

Puede instalar paquetes adicionales especificándolos después del campo Write additional packages to install ("Escriba paquetes adicionales para instalar").

Una vez se complete la instalación, un texto verde debería aparecer indicando que es seguro reiniciar (reboot) el sistema, el cual es, a su vez, el comando que debe utilizar para reiniciar.

Perfiles

archinstall incluye perfiles, o conjuntos de paquetes y opciones pre-configuradas que pueden ser instaladas junto al sistema operativo base.

Advertencia: Los perfiles suministrados son específicos de archinstall y no están soportados por los mantenedores del paquete. Se recomienda a los usuarios comprobar los detalles de cada perfil antes de utilizarlo.
