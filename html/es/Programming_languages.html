Esta página intenta listar todos los lenguajes de programación con un compilador / intérprete empaquetado para Arch Linux.

  • Ensamblador – as de binutils, fasm, nasm, yasm
  • Ada – GCC
  • AWK
  • BASIC – freebasic
    • AOZ Studio
    • Gambas
  • C – GCC, Clang
  • C++ – GCC, Clang
  • C# – Mono, .NET Core
    • Objective-C – GCC, Clang
    • Vala – vala
    • Zig – zig
  • Crystal
  • D
  • Dart – dart
  • FIM++ – fimpp-gitAUR[enlace roto: package not found]
  • Forth – 4thAUR, gforthAUR
  • Fortran – GCC
  • Go – go, GCC
  • Java
    • Groovy – groovy
  • JavaScript – rhino, v8AUR
    • Node.js
  • Julia
  • Kotlin – kotlin
  • Lua – lua
  • Nim – nim
  • Octave
  • Pascal – fpc
  • Perl
  • PHP
  • Python
  • R
  • Ruby
  • Rust
  • Scala
  • Swift
  • Tcl – tcl

Lenguajes funcionales

Tenga en cuenta que algunos de los lenguajes mencionados anteriormente también admiten alguna programación funcional.

  • Erlang – erlang
    • Elixir
  • Haskell
  • Lisp dialects
    • Clojure
    • Common Lisp
    • Emacs Lisp
    • Scheme
    • racket - racket, racket-minimal
  • ML
    • Standard ML – smlnj, mlton, polyml, mosmlAUR, smlsharpAUR
    • OCaml – ocaml
