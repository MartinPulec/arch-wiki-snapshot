De Ext4 - Novatos del kernel de Linux:

Ext4 es la evolución del sistema de archivos Linux más utilizado, Ext3. En muchos sentidos, Ext4 es una mejora más profunda con respecto a Ext3 que Ext3 con respecto a Ext2. Ext3 se trataba principalmente de agregar registros en diario a Ext2, pero Ext4 modifica estructuras de datos importantes del sistema de archivos, como las destinadas a almacenar los datos del archivo. El resultado es un sistema de archivos con un diseño mejorado, mejor rendimiento, confiabilidad y características.

Crear un nuevo sistema de archivos ext4

Instala e2fsprogs.

Para formatear una partición haga: 

# mkfs.ext4 /dev/partition
Tip:
  • Consulte mke2fs(8) para obtener más opciones; edite /etc/mke2fs.conf para ver/configurar las opciones predeterminadas.
  • Si es compatible, es posible que desee habilitar metadata checksums[enlace roto: sección no válida].
