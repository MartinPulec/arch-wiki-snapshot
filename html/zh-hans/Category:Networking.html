From ArchWiki

Arch Linux 中可以用netctl进行网络配置。netctl同时支持有线网络和无线网络、网络漫游，简化了网络配置文件的管理. NetworkManager也是流行的替代工具。

与连接网络、提供鉴权、或提供特殊的(非最终用户的)服务的程序通常可在此类别找到。

点对点(P2P)网络、BT客户端、聊天客户端，和其他的 internet-aware 应用程序可以在类别 Internet Applications找到。提供 web 页面、邮件服务、和代码仓库服务的软件可以在类别Web Server找到。内容管理系统(CMS)在 Web Server 下列出，但是它们也有大型数据库组件。数据库服务不在 web servers 中列出，参见 Database management systems.
