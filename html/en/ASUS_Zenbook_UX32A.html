This page contains information and tips for installing and configuring Arch Linux on the Asus Zenbook UX32A.

Installation

The installation should work with no problems and wifi should work out of the box. The UX32A ships in two versions: either with 500 GB HDD + 24 GB SSD or with 128 GB HDD and 128 GB SSD. Since 24 GB is more then enough for a full Arch Linux installation, it is recommended to install at least the non-home partitions onto the SSD.

Warning: This could lead to problems if Windows stays installed on the same machine.

Compatibility

Most things work without problems. To control the keyboard backlight, you have to follow the instructions here: ASUS Zenbook Prime UX31A#Keyboard backlight.

Function keys

The following function keys work:

  • Standby (Fn+F1)
  • Screen brightness (Fn+F5/F6)
  • Turn Screen on/off (Fn+F7)

The other keys are recognised and yield the specific signal with the exception of the Presentation Mode (Fn+F8), which only yields 'p' and the wifi button, which is not recognised at all.
