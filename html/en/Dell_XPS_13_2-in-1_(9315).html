Installation

RAID mode is enabled by default. AHCI mode must be used, otherwise the disks will be invisible [1]. Using RAID mode will trigger a relevant log message in the journal.

Accessibility

To enter the BIOS setup press Volume down on the keyboard, otherwise for the boot menu press the End key.

For more information it is advisable to visit the service manual.

Audio

ALSA firmware is required to make audio work.

Accelerometer and ambient-light sensor

iio-sensor-proxy is required to work.

Stylus pen (PN9315A)

Works out of the box for writing under Wayland, after configuring the bluetooth service. See Bluetooth, libinput and libwacom.

Note: The button on top of the pen works via Bluetooth. The other buttons do not work via Bluetooth.

This button will be seen by the system as an additional key, so by customizing the keyboard shortcuts it will be possible to associate a function to it via the ydotool package.

Xorg needs Bluetooth, xf86-input-libinput, xf86-input-wacom and xdotool to associate and execute the shortcut to activate a window by pressing the pen's top button.

Function keys

Function keys have two options in the BIOS. By default the Fn lock is enabled, so pressing Fn triggers the F1, F2 etc. keys; by changing the configuration in the BIOS you can have the media keys as primary instead.

Otherwise by pressing Fn+Esc you can lock or unlock the Fn key.
