Lenovo Legion Go

From ArchWiki

This article or section does not follow the Laptop page guidelines.

Reason: Stub: this page is incomplete. You can help by expanding it! (Discuss in Talk:Lenovo Legion Go)

The Lenovo Legion Go is a gaming-focused PC handheld by Lenovo. It features a AMD Ryzen Z1 Extreme processor, LPDDR5x-7500 shared memory, and a Radeon 780M with 12 CUs and up to 2.7 GHz clock speed.

Installation

The device comes preloaded with Windows. It can be used to update the BIOS via a Installer File before installing Arch. Disable Secure Boot in the BIOS.
