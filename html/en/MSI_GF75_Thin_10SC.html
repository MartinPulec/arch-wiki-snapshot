Firmware

Press F2 to enter BIOS menu.

You can swap Fn and Super keys.

Secure Boot can be disabled from the Security tab.

Function keys

Key Visible? Marked? Effect
Fn+F2 Yes Yes Output switching
Fn+F3 No Yes
Fn+F4 No Yes
Fn+F5 No Yes
Fn+F6 Yes Yes Switch state of webcam
Fn+F7 No Yes
Fn+F10 No Yes
Fn+F12 Yes Yes XF86Sleep
Fn+Left Yes Yes XF86AudioLowerVolume
Fn+Right Yes Yes XF86AudioRaiseVolume
Fn+Up Yes Yes XF86MonBrightnessUp
Fn+Down Yes Yes XF86MonBrightnessDown
Fn+Num_0 Yes Yes XF86AudioMute
Fn+Num_Minus No Yes Raise keyboard backlight
Fn+Num_Plus No Yes Decrease keyboard backlight

Webcam

Webcam works out of the box with the uvcvideo kernel module.

Note: Do not forget to turn on the webcam with Fn+F6!
