Fingerprint reader

The fingerprint reader is not currently supported by libfprint link. Install libfprint-fpcmoh-gitAUR to add the proprietary Lenovo driver to libfprint.

Function keys

Both the function lock and Fn+F1 (XF86AudioMute) have indicator lights.
