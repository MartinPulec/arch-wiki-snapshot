Accessibility

The appearance of the BIOS is pretty simple (grey and blue), so it might work well with OCR software. It does not have mouse support

Note: Blind users should request the help of a sighted person to change BIOS settings

Firmware

fwupd does not support this device yet.

GPU

The NVIDIA GPU / AMD iGPU combo supports PRIME
