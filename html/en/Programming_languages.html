This page attempts to list all programming languages with a compiler / interpreter packaged for Arch Linux.

  • Assembly – as of binutils, fasm, nasm, yasm
  • Ada – Ada
  • AWK
  • Ballerina - ballerinaAUR
  • BASIC – freebasic
    • AOZ Studio
    • Gambas
    • QB64 – qb64AUR
  • C – GCC, Clang
    • C++ – GCC, Clang
    • Objective-C – GCC, Clang
  • C# – Mono, .NET Core
  • Crystal
  • D
  • Dart – dart
  • Erlang – erlang
    • Elixir
  • Factor – factorAUR
  • Forth – 4thAUR, gforthAUR, retroforthAUR
  • Fortran – GCC
  • Go – go, GCC
  • Java
    • Groovy – groovy
  • GDL - gnudatalanguageAUR
  • Haskell
  • JavaScript – rhino, v8AUR[broken link: package not found], quickjsAUR
    • Node.js
  • Julia
  • Kotlin – kotlin
  • Lisp dialects
    • Clojure
    • Common Lisp
    • Emacs Lisp
    • Scheme
    • racket - racket, racket-minimal
    • newLISP – newlispAUR
    • PicoLisp – picolispAUR
  • Lua – lua, luajit
  • ML
    • Standard ML – smlnj, mlton, polyml, mosmlAUR, smlsharpAUR
    • OCaml – ocaml
  • Nim – nim
  • Octave
  • Pascal – fpc, lazarus
  • Perl
  • PHP
  • Python
  • QCL – qclAUR
  • R
  • Raku (Perl 6) – rakudoAUR
  • Ruby
  • Rust
  • Scala
  • Swift
  • Tcl – tcl
  • Vala – vala
  • Zig – zig
