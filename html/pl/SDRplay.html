SDRplay jest producentem szeregu odbiorników radiowych definiowanych programowo (SDR) opartych na technologii Mirics Semiconductor. MSi2500 który jest używany w różnych aplikacjach, od DVB-T po AM Radio.

Sterownik

Sterownik SDRplay jest zamknięto-źródłowy, więc jest dostępny tylko w formacie binarnym: można go pobrać lub najlepiej zainstalować z pakietu libsdrplayAUR.

SoapySDR

Niewiele aplikacji obsługuje obecnie libmirsdrapi-rsp, więc dobrym pomysłem jest zainstalowanie soapysdr lub soapysdr-gitAUR.

The SoapySDRPlay module is also required and is available from soapysdrplay3-gitAUR for the v3 API (compatible with the latest soapysdr), or soapysdrplay-gitAUR[broken link: package not found] for the v2 one.

GNU Radio

Użytkownicy chcący używać SDRplay z GNU Radio będą musieli zainstalować gr-osmosdr-nonfree-gitAUR[broken link: package not found].

Inne aplikacje

  • cubicsdrAUR zapewnia w pełni funkcjonalną i łatwą w użyciu aplikację SDR.
  • gqrxAUR kolejna łatwa w użyciu aplikacja SDR zbudowana na GNU Radio.
  • sdrangel-binAUR lub sdrangel-gitAUR wszechstronne narzędzie z ładnie wyglądającym interfejsem, może również dekodować głos cyfrowy
