Wykonaj następujące kroki, aby użyć Energia dla płytki Launchpad MSP-EXP430G2.

Zainstaluj pakiet java-rxtx.

Stwórz regułę udev: 

/etc/udev/rules.d/10-launchpad.rules
SUBSYSTEM=="usb",ATTR{idVendor}=="0451",ATTR{idProduct}=="f432",GROUP="dialout",MODE="0660"

Stwórz grupę dialout: 

# groupadd dialout

Dodaj twojego użytkownika do grup lock, uucp oraz dialout: 

# usermod -a -G lock,uucp,dialout $(whoami)

Pobierz energię ze strony www.energia.nu i zdekompresuj ją.

Zmień katalog roboczy na folder lib: 

$ cd ./energia-0101E0012/lib/

Utwórz kopię zapasową tych plików (lub usuń je): 

$ mv librxtxSerial.so librxtxSerial.so.bak
$ mv librxtxSerial64.so librxtxSerial64.so.bak
$ mv RXTXcomm.jar RXTXcomm.jar.bak

Stwórz linki symboliczne do wersji Arch Linuxowej pakietu java-rxtx: 

$ ln -s /usr/lib/librxtxSerial.so "librxtxSerial.so"
$ ln -s /usr/lib/librxtxSerial.so "librxtxSerial64.so"
$ ln -s /usr/share/java/rxtx/RXTXcomm.jar "RXTXcomm.jar"

Uruchom ponownie komputer (lub zrestartuj udev, wyloguj się i zaloguj).

Podłącz launchpad i uruchom program Energia.
