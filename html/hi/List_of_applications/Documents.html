Translation and localization (अनुवाद और स्थानीयकरण)

Wikipedia:Comparison of computer-assisted translation tools भी देखें।

  • Apertium — Available language data के साथ free और open source rule-based machine translation platform। यह निम्नलिखित formats को supports करता है: HTML, Microsoft Office 2007 XML, OpenDocument, TMX, MediaWiki और others।
https://www.apertium.org/ || apertiumAUR
  • Crow Translate — Google, Yandex और Bing का उपयोग करके text को translate और speak करने की अनुमति देने वाला simple और lightweight translator।
https://apps.kde.org/crowtranslate/ || crow-translate
  • Dialect — Google Translate और अन्य online services पर आधारित GNOME के लिए translation app।
https://apps.gnome.org/Dialect/ || dialect
  • Gtranslator — GNOME के लिए enhanced gettext po file editor। यह gettext po files के सभी forms को handles करता है और बहुत useful features शामिल करता है।
https://wiki.gnome.org/Apps/Gtranslator || gtranslator
  • Lokalize — Software translation के लिए standard KDE tool। इसमें PO files की basic editing, glossary के लिए support, translation memory, project managing आदि शामिल है। kde-sdk का हिस्सा।
https://apps.kde.org/lokalize/ || lokalize
  • Moses — Statistical machine translation tool (language data शामिल नहीं है)।
http://statmt.org/moses/ || mosesdecoderAUR
  • OmegaT — General translator's tool जिसमें बहुत सारे translation memory features हैं और Google Translate से suggestions दे सकता है। यह निम्नलिखित formats को supports करता है: HTML, Microsoft Office 2007 XML, OpenDocument, XLIFF/Okapi, MediaWiki, plain text, TMX और others।
https://omegat.org/ || omegatAUR
  • Poedit — Gettext (PO, POT) और XLIFF के लिए simple translation editor।
https://poedit.net || poedit
  • Pology — Gettext/po-files के साथ dealing करने के लिए Python tools का set।
https://techbase.kde.org/Localization/Tools/Pology || pology-gitAUR
  • Qt Linguist — Qt C++ और Qt Quick applications को local languages में translate करना।
https://doc.qt.io/qt-6/qtlinguist-index.html || qt6-tools
  • Translate Shell — Google Translate के लिए command-line interface और interactive shell।
https://www.soimort.org/translate-shell/ || translate-shell
  • Translate Toolkit — Localization और translation toolkit, जो localization file formats और files जिन्हें localization की आवश्यकता हो सकती है के साथ काम करने के लिए tools का set प्रदान करता है।
https://toolkit.translatehouse.org/ || translate-toolkit
