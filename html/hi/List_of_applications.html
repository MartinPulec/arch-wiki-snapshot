यह article एक सामान्य applications की list है जो category के अनुसार sort की गई है, उन लोगों के लिए reference के रूप में जो packages खोज रहे हैं। कई sections console और graphical applications के बीच split हैं। "Console" sections में listed applications के graphical front-ends हो सकते हैं, official ones को currently omit किया गया है।

Tip
  • कृपया इंस्टॉल करने पर विचार करें pkgstats package, जो एक timer प्रदान करता है जो आपके system पर installed packages की list, architecture और mirrors के साथ, Arch Linux developers को भेजता है ताकि उन्हें अपने efforts को prioritize करने और distribution को और बेहतर बनाने में मदद मिल सके। यह information anonymously भेजी जाती है और आपको identify करने के लिए use नहीं की जा सकती। आप collected data को Statistics page पर देख सकते हैं। अधिक information इस forum thread में उपलब्ध है।
  • Daemon packages में आमतौर पर relevant systemd unit file स्टार्ट करने के लिए शामिल होती है; कुछ packages में अलग-अलग भी होती हैं। Installation के बाद pacman -Qql package | grep -Fe .service -e .socket का उपयोग relevant file को check और find करने के लिए किया जा सकता है।
