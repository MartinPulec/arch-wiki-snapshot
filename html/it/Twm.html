  • Window manager (Italiano)

twm è un window manager per Xorg. È un programma piccolo, che utilizza Xlib invece di una libreria di widget, e quindi richiede poche risorse di sistema. Anche se semplice, è altamente configurabile: caratteri, colori, larghezza dei bordi, barra del titolo etc. possono essere impostati dall'utente.

twm è stato scritto da Tom LaStrange, uno sviluppatore che era frustrato dalle limitazioni di uwm (Ultrix Window Manager), unico gestore di finestre per X11 al momento del suo rilascio. [1]

TWM ha sostituito uwm come window manager predefinito di X11 dalla pubblicazione di X11R4 nel 1989.[2]

Negli anni l'acronimo twm ha indicato Tom's Window Manager, Tab Window Manager e più recentemente di Timeless Window Manager.

Installazione

Installare xorg-twm, disponibile nei repository ufficiali.

Avvio

Eseguire twm con Xinit.

Note: All'avvio viene mostrato solamente uno schermo nero. Muovere il mouse e cliccare con il tasto sinistro per mostrare un menù di twm ed accertarsi che twm funzioni.

È possibile avviare twm tramite un display manager. Il file twm.desktop non esiste, quindi deve essere creato al percorso /usr/share/xsessions/. Nel nuovo file /usr/share/xsessions/twm.desktop inserire: 

/usr/share/xsessions/twm.desktop
[Desktop Entry]
Name=twm
Comment=xorg-twm
TryExec=twm
Exec=twm
Type=Application

Configurazione

Per impostazione predefinita, twm appare molto datato e poco intuitivo. Modificando il file ~/.twmrc è possibile personalizzare twm per renderlo più user-friendly.

La pagina twm(1) spiega in dettaglio quali comandi possono essere utilizzati nel file ~/.twmrc.

Trucchi e consigli

Versione modificata

Esiste una versione modificata, non presente nei repository, con funzionalità aggiornate come la trasparenza. Una descrizione ed uno script di compilazione sono disponibile sulla mailing list di xorg. Può essere provato installando xcompmgr, eseguendo lo script di compilazione, mettendo i file risultanti twm e dot.twmrc nelle cartelle apposite, e modificando il file ~/.xinitrc in modo che le ultime due righe siano: 

xcompmgr -o 0.3  -c -r 8 -t -10 -l -12 &
/path-to-directory/twm -visual TrueColor -depth 32 -f /path-to-directory/dot.twmrc

Risoluzione di problemi

Barre del titolo e menù troppo grandi

Può capitare che le barre del titolo ed i menù di twm siano troppo larghi - il doppio di quanto ci si aspetterebbe. È un problema di localizzazione che capita quando viene utilizzata una localizzazione UTF-8. Per risolvere il problema impostare la localizzazione a C. Vedere anche [3].
