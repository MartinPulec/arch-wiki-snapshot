Xournal++ (Italiano)

  • English
  • 日本語
Appearance
From ArchWiki

Xournal++ è un app open source per la scrittura a mano, scritta in C++, che supporta l'annotazione dei file PDF, è una riscrittura di Xournal, per essere più efficiente e migliorare la funzionalità, pur rimanendo retrocompatibile con Xournal, quindi in grado di leggere e modificare i suoi file (.xoj).

Installazione

Xournal++ è fornito in più pacchetti, installare uno dei due:

  • xournalpp - Ultimo rilascio stabile.
  • xournalpp-gitAUR - Ultimo git commit, compilato da sorgente.

Risoluzione dei problemi

Perché lo sfondo del PDF è scomparso dopo aver spostato i file?

Xournal++ attualmente non supporta l'inclusione del PDF nei file, quindi sia il PDF che il file .xopp devono essere tenuti insieme altrimenti lo sfondo del PDF scomparirà. Ci stiamo lavorando, è possibile trovare il problema su Github.

Xournal può leggere i file Xournal++ (.xopp)?

In teoria sì, secondo gli sviluppatori, il formato del file è lo stesso, tuttavia .xopp memorizza nuove funzionalità che non sono incluse in Xournal, quindi finché solo le funzionalità sono supportate sia in Xournal che in Xournal++, il file dovrebbe essere intercambiabile tra le applicazioni.

Tuttavia, Xournal++ può esportare file .xopp come file .xoj per convertire i file da Xournal++ a Xournal.

Impossibile avviare la registrazione

Controlla se il dispositivo impostato nelle preferenze di Xournal++ è un microfono effettivo [1].
